# Prompt Pack

Use these templates when Codex assigns stable work to Claude Code workers.

Pick one template:

- `repo-audit.md`: inspect a repo and find risks.
- `bugfix.md`: fix one bug inside a write scope.
- `security-audit.md`: audit secrets, auth, permissions, and dangerous actions.
- `frontend-polish.md`: improve a UI without breaking behavior.
- `test-generation.md`: add focused tests.
- `refactor-plan.md`: plan a refactor before writing code.
- `release-check.md`: check readiness before publish.

Controller rule: Codex owns the final decision. Worker output is evidence.
